About Git Vital

What is Git Vital

Git Vital is a repository analyser using a classifier trained from over 90,000 randomly selected repositories on Github.com. The classifiers are built using Weka with the PART algorithm after comparing other classifiers and their performance.

How does Git Vital work?

Git Vital mines the Github API for information on a repository and sends it to a classifier server to be processed. It compares various parameters such as the number of contributors to the size of the repository.
